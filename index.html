What is jQuery-ace?

It's a jQuery plugin which allows to embed ACE Code Editor into any chosen div or textarea

How to install and use

First you need to include jQuery and Ace libraries. I provide it's bundles in archive in case that you haven't them. Then just include plugin source script: {% highlight html %} {% endhighlight %} After that apply plugin to the existing textarea: {% highlight html %} {% endhighlight %} And ACE Code Edtior will be embeded immediately:

Rails integration

If you are using Rails then you could just add jquery-ace-rails gem to Gemfile: {% highlight ruby %} gem 'jquery-ace-rails' {% endhighlight %} And include scripts using sprockets in your application.js: {% highlight javascript %} //= require jquery/jquery-1.8.3.min //= require ace/ace //= require ace/theme-twilight //= require ace/mode-ruby //= require jquery-ace.min {% endhighlight %}

Access to ACE Code Editor instance

Plugin stores instance of the ACE Code Editor to the elements data. You can access to it through ace data key. {% highlight javascript %} // Decorator var decorator = $('.my-code-area').data('ace'); // ACE Code Editor instance var aceInstance = decorator.ace; {% endhighlight %}
